🇵🇹 PortoGo — AI Assistant for Relocating to Portugal

Live App: https://portogo-mdandcpdthrvtcsvmwmvzm.streamlit.app/

What this is

PortoGo is an experimental AI assistant designed to help people navigate the often confusing process of relocating to Portugal.

It acts like a conversational guide. Instead of digging through scattered articles, users can ask questions in plain language and get structured, relevant answers about visas, documents, and next steps.

Why I built this

Relocating to another country is overwhelming. Information is fragmented, requirements change, and official sources are not always easy to interpret.

As someone with a background in UX and research, I wanted to explore a simple idea:

What if relocation guidance felt more like a conversation than a search problem?

PortoGo is my way of testing how AI can reduce cognitive load and help people move forward with more clarity and confidence.

What it does
Answers questions about moving to Portugal in natural language
Provides guidance on visa types and requirements
Helps users understand next steps based on their situation
Reduces the need to navigate multiple sources of information
Offers a lightweight, conversational experience instead of static content
How it works

The app is built as a conversational interface powered by AI.

Users type questions like:

“Which visa should I apply for?”
“What documents do I need?”
“How long does the process take?”

The system interprets intent and returns structured, relevant guidance.

Tech stack
Python
Streamlit
OpenAI API
Prompt design for conversational UX
Design thinking behind it

This project is less about complex engineering and more about interaction design.

Some of the principles I focused on:

Reducing cognitive load
Breaking down complex processes into digestible responses
Conversational clarity
Making answers feel human and easy to follow
Progressive disclosure
Giving users what they need without overwhelming them
User intent over rigid flows
Letting people ask questions instead of forcing navigation paths
What I’d improve next

This is an evolving experiment. If I continued building it, I would:

Add personalized user profiles and saved progress
Improve accuracy with structured data sources
Introduce step-by-step relocation plans
Add document checklists tailored to each visa type
Design a more guided onboarding experience
